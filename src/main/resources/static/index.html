VUTTR API(Very Useful Tools to Remember)

Repositório para gerenciar ferramentas com seus respectivos nomes, links, descrições e tags

Resources:
  • GET : "/tools"
  • GET : "/tools?tag={tag}"
  • POST : "/tools"
  • DELETE : "/tools/{id}"
  • GET : "/tags"
  • GET : "/tags/{id}"

Acesse a documentação completa da API criada com o Swagger
